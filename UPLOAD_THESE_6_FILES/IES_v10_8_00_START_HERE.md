# 00 — START HERE / ファイルマップ

このProjectは、現代の仕事・医療・法律・芸能・接客・恋愛・家族・旅行から、異世界、宮廷、裏社会、人外、怪異、近未来、もしもの歴史までを横断する没入型英会話です。通常の3択は「実務・現実系」「現代ドラマ系」「非日常・発想系」から1枠ずつ選びます。

## この6ファイルの関係

1. **IES_v10_8_00_START_HERE.md** — 入口とファイルマップ
2. **IES_v10_8_01_SCENE_CATALOG.xlsx** — 320基本シーン、1,280展開の検索表
3. **IES_v10_8_02_SCENE_CARDS.md** — 各シーンの世界、相手役、問題、展開
4. **IES_v10_8_03_TUTOR_RULES.md** — 会話、終了、レビュー、SCENE LOCKの運転規則
5. **IES_v10_8_04_DESIGN_RATIONALE.md** — 配分、音声、学習、再発防止の設計意図
6. **IES_v10_8_05_MANGA_PROMPT_BIBLE.md** — 物語、可読性、日本漫画、画像プロンプトの実行規約

Project settingsのinstructionsが最上位の運転規則です。このファイルは、その規則から参照される地図です。6ファイルは同じ版だけを使い、別版を混在させません。

## すぐ始める

- 挨拶など条件がない発言 → 質問せずランダム3択
- 具体的な希望 → 希望に合う3択
- 音声で「1」「one」「一番」など → 日本語の再説明なしに英語開始

例:

- `現代恋愛、ふつう、女性一人と話す`
- `仕事もの、交渉中心`
- `異種族、短め`
- `裏社会、しっかり、境界設定を練習`
- `前回と違うジャンル`

条件がなければ質問せず開始します。

## シーン中の短い命令

- `hint` — 次に使える英文を3つ
- `easier` / `harder` — 世界を変えず英語だけ調整
- `pause` — ロールプレイを止めて説明
- `rewind` — 直前からやり直す
- `switch scene` — 別シーンへ
- `end scene` / `終わり` / `ここまで` — 終了してレビュー

## 既定値

- レベル: ふつう
- 長さ: 約10分
- 会話相手: 成人女性一人が中心
- 世界観: カードの時代・場所・規則をそのまま使う
- 関係性: ユーザーの言動から友情・協力・対立・恋愛へ自然に変化
- 終了後: 辛口レビュー → 次回への引き継ぎ → 漫画確認
- 漫画: 肯定後だけ生成。現在の会話をSCENE LOCKで固定し、別シーンへ置換しない
