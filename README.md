# 没入型英会話 — Immersive English Scenes

ChatGPT Project専用の英会話エンジン。一言目から相手役が動き出し、いきなり英語の会話に入ります。

## これは何か

定型フレーズの暗記や単調な一問一答ではなく、320本のシーン(1,280通りの展開)の中から選んだ「相手役」と、状況の続きとして英語を使います。相手役は成人女性一人に固定され、人格・声・世界観を保ったまま、実務の交渉から異世界の危機まで幅広い状況を扱います。汎用のAI英会話練習とは、次の点が違います。

## 何が違うのか

- **状況の幅が本物**: 現代の仕事・医療・法律・接客・恋愛・家族・旅行から、異世界・宮廷・裏社会・人外・怪異・近未来まで、320基本シーン×4展開=1,280通り。希望やタイトルを指定しない通常時は、3択が「実務・現実系」「現代ドラマ系」「非日常・発想系」から1枠ずつ選ばれるので、ビジネス英語だけにも非日常のロールプレイだけにも偏りません(具体的な希望やタイトルを伝えると、その指定が抽選より優先されます)。
- **相手役が会話を主導する**: 成人女性一人の相手役が状況・答えやすい質問・反応・次の展開を出して進めます。学習者が会話の設計や質問役を背負う必要はありません。
- **難易度は世界を変えない**: やさしい/ふつう/しっかり/挑戦の4段階は、語彙・文長・含意・ヒント量だけを調整します。時代・場所・危険度・種族といった舞台設定はレベルに関係なく保たれます。
- **辛口レビューは1本の「覚える英文」に絞る**: 会話が自然に終わると日本語で辛口レビューが始まります。改善点を大量に並べるのではなく、最大の改善点を1つの英文(HERO LINE)として提示し、自然な日本語訳を添えます。補助的な改善は1〜2件だけです。
- **学習履歴はメニューの1枠だけに反映**: 過去の傾向は、次回メニュー3択のうち1枠と難易度調整にだけ使われます。残り2枠は新しいジャンルを優先し、弱点ばかりを繰り返し提示しません。

## 会話の流れ

1. 最初のひとことを送る。挨拶だけでもよいし、`現代恋愛、ふつう、女性一人と話す` `異種族、しっかり、境界設定を練習` のような希望でもかまいません。
2. 3つの日本語の選択肢が出ます。声でもテキストでも「1」「2」「3」と選ぶと、確認や日本語の再説明を挟まず、そのまま英語の会話が始まります。
3. 相手役と英語でロールプレイが進みます。`hint`(次に使える英文)、`easier` / `harder`(世界を変えず英語だけ調整)、`pause`(説明モード)、`rewind`(やり直し)、`switch scene`、`end scene` / `終わり` などの短い命令はいつでも使えます。
4. 会話が自然に終わると、相手役の短い締めのセリフ → 🔥辛口レビュー(HERO LINE付き) → 📌次回への引き継ぎ、の順で表示されます。
5. 最後に「今回の改善表現を入れた1ページ漫画を作りますか？」と聞かれます。ここで明確に「はい」と答えたときだけ漫画が生成されます。

## 教育的漫画について

「はい」と答えると、その会話で固定された人物・関係・時代・場所・世界観と、レビューで示したHERO LINEだけを使って、ペン入れ・スクリーントーンなど本格的な日本漫画の技法で、スマホで読み切れる1ページが生成されます。

ただし漫画は**会話の再現や要約ではありません**。同じ状況設定の中で、HERO LINEを言わなければ収まらない新しい小さな事件をその場で発明し、その一言で予想外に展開が変わる、記憶に残る短いエピソードとして描かれます。ページ内の文字はHERO LINEのみで、コマ数は内容に応じて3〜6コマの範囲で自動調整されます。

「いいえ」と答えれば、漫画を作らずに静かに終了します。

## インストール

1. ChatGPTで新しいProjectを作成します。作成時に選べる**メモリ設定は「このProjectのみ」を選ぶことを推奨**します。このProjectは同じProject内のチャットをまたいだ継続性(学習履歴・次回への引き継ぎ)を前提に設計されているため、Project専用メモリにすると、無関係な他のチャットの情報を持ち込まずに済みます。名前は自由です(バージョン番号などを入れる必要はありません)。
   - Project専用メモリは**新規Project作成時**にだけ選べます。すでに既定のメモリ設定で作ったProjectを後から切り替えることはできないため、その場合は新しいProjectを作り直してください。
   - Project専用メモリは同じProject内のチャットだけを参照し、Project外の保存済みメモリや他のチャットは参照しません。利用可否はアカウント/ワークスペースのメモリ設定にも依存します。
   - 詳細は公式ヘルプ(下記)を参照してください。
2. Project settingsのinstructionsへ `PROJECT_INSTRUCTIONS.txt` の全文を貼ります。
3. Project sourcesへ `UPLOAD_THESE_6_FILES` 内の6ファイルを**すべて**追加します(一部だけの追加では正しく動作しません)。この6ファイルを収めるには、Project内ファイル数の上限が6以上あるプラン/ワークスペースが必要です(公式ヘルプ記載時点でFreeプランの上限は5ファイルのため不足します)。上限は変更されることがあるため、最新の値は公式ヘルプ(下記)で確認してください。
4. 新しいチャットで、挨拶または希望を一言送ります。
