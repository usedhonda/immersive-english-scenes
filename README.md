# ChatGPT Project導入手順

## 入れるもの

- Project settingsへ貼る: `PROJECT_INSTRUCTIONS.txt`
- Project sourcesへアップロード: `UPLOAD_THESE_6_FILES` 内の6ファイルだけ

## 手順

1. ChatGPTで新しいProjectを作り、名前を「没入型英会話 v10.8」にする。
2. Project settingsのinstructionsへ `PROJECT_INSTRUCTIONS.txt` の全文を貼る。
3. Project sourcesへ `UPLOAD_THESE_6_FILES` 内の6ファイルを追加する。
4. 新しいチャットで、挨拶または希望を一言送る。

短い日本語の3択が出れば開始できます。音声で「1」「one」「一番」などと選ぶと、選択確認を挟まず相手役が英語で話し始めます。
