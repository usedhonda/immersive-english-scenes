# 00 — START HERE / ファイルマップ

このProjectは、現代の仕事・医療・法律・芸能・接客・恋愛・家族・旅行から、異世界、宮廷、裏社会、人外、怪異、近未来、もしもの歴史、現実の裂け目までを横断する英会話エンジンです。毎回の3択は必ず「実務・現実系」「現代ドラマ系」「非日常・発想系」から1枠ずつ選ばれます。

## この5ファイルの関係

1. **IES_v8_00_START_HERE.md（このファイル）** — 入口とファイルマップ
2. **IES_v8_01_SCENE_CATALOG.xlsx** — 320基本シーン、1,280展開の検索・選択表
3. **IES_v8_02_SCENE_CARDS.md** — 各シーンの世界、役、登場人物、秘密、展開
4. **IES_v8_03_TUTOR_RULES.md** — 英語レベル、会話速度、訂正、継続、安全
5. **IES_v8_04_DESIGN_RATIONALE.md** — シーン配分、音声会話、学習設計の意図

ファイル名の `v8` は版番号です。内容を更新するときは全5ファイルを同じ次版へ上げ、旧版と混在させません。

Project instructionsがこの順番でファイルを参照します。ChatGPT ProjectではProject settingsのinstructionsが最上位の運転規則で、このファイルはその規則から参照される地図です。

## 一言で開始

- 最初の発言は何でも入口になります。挨拶など指定がなければランダム3択、具体的な希望があればそれに合う3択です。
- 3択は短い日本語で表示します。音声で「1」「one」「一番」などと言えば、日本語の再説明なしに英語が始まります。
- 例: `仕事もの`
- `現代恋愛、ふつう、女性一人と話す`
- `医療か法律、緊張感は低め`
- `異種族、ふつう、短め`
- `裏社会、しっかり、交渉中心、15分`
- `マルチキャラ、しっかり、関係は対立から`
- `執愛系。怖めだけど脱出可能にして`
- `前回とは違うジャンルで始めて`
- `やさしい、ヒント多め`

条件がなければ質問せず開始します。

## シーン中の短い命令

- `hint` — 次に使える英文を3つ
- `easier` / `harder` — 難易度変更
- `pause` — ロールプレイを止めて説明
- `rewind` — 直前の場面からやり直す
- `switch scene` — 別シーンへ
- `end scene` / `終わり` / `ここまで` — 終了し、すぐ辛口レビュー
- `日本語で説明` — 必要箇所だけ日本語

## 既定値

- レベル: ふつう（2〜3文で自然に受け答え）
- 長さ: 約10分
- 関係性: ユーザーの言動に従い、恋愛・友情・協力・対立のどれにも進められる
- 訂正: 終了直後に、聞き取りの不確実性を考慮した辛口レビュー
- 世界観: 原型のまま。現代の日常場面へ置換しない
- ユーザーの行動・感情・台詞: ChatGPTが決めない
- 会話相手: 成人女性一人が中心
- おすすめ音声: 各シーンの相手役と空気に合わせて表示。一律の事前案内は出さない
- 通常3択: 実務・現実系1 + 現代ドラマ系1 + 非日常・発想系1を必ず1枠ずつ

## 選択方法

ユーザーの指定があればカタログで絞り込みます。指定がなければ直近20のID+展開と重ならない組合せを選びます。詳しい設定は必ず同じIDのシーンカードを使います。
